# Operacao -de-registro-de-live-multi-loja
 Sistema web em Google Apps Script para gestão de vendas ao vivo multi loja. Permite registrar vendas por live, controlar apresentadoras, acompanhar resultados em tempo real, gerenciar histórico e exportar relatórios. Interface moderna, rápida e otimizada, com foco em simplicidade, controle e desempenho.
